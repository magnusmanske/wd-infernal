Wikidata Infernal

Wikidata Infernal is an API that allows you to infer new facts from Wikidata. It uses a set of rules to infer new facts from existing ones. The generated statements will have qualifiers to indicate the source and method of the inference.

  • /P131/LAT/LON

    Finds items with a P131 near the given location, and returns their administrative region (P131) as statements.

    Example (a point in Cambridge, UK)
