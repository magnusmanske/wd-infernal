Wikidata Infernal

Wikidata Infernal is an API that allows you to infer new facts from Wikidata. It uses a set of rules to infer new facts from existing ones. The generated statements will have qualifiers to indicate the source and method of the inference. Output is an array of statements in JSON/Wikidata format.

  • /P131/LAT/LON

    Finds items with a P131 near the given location, and returns their administrative region (P131) as statements.

    Example (a point in Cambridge, UK)

  • /name_gender/NAME

    Parses a (western-style, "first_name last_name") personal name, and tries to infer items for first and last names, as well as gender, based on first names.

    Example (Yours Truly)

  • /country_year/ITEM/YEAR

    /country_year/ITEM/YEAR/PROPERTY

    For a given location item (e.g. a city), finds the country in the given year, and returns it as a statement. Will return the "default" country (no qualifiers) if no better match can be found.

    Example (country of citizenship for people born in Cologne in 1921)

  • /cross_categories/CATEGORY_ITEM/LANGUAGE/DEPTH

    For a given Wikidata item that is for a category, finds all pages with the category on the sitelinked wikis (to depth), the items for these pages, and returns the pages for the items matching the language wiki (not using disambigution pages, or pages already in the language wiki category).

    Example (Category:Polish skydivers for en.wikipedia.org, depth 5)

  • /referee/ITEM

    For a given Wikidata item, searches linked external IDs, official website, and all web pages linked from associated Wikipedia articles for potential references to statements in the item. It tries to use different date formats, and the respective label for the language of the website.

    Example (Carlo Creazzo, random example)

  • /isbn/isbn/ISBN

    For a given ISBN, searches ISBN sources and generates a Wikibase REST API POST payload for a new item.

    Example (Le seigneur des anneaux T1 La fraternit√© de l'anneau, by JRR Tolkien)

  • /isbn/item/ITEM

    For a given Wikidata item with an ISBN statement, searches ISBN sources and generates a Wikibase REST API patch for the item.

    Example (Caballeros de fortuna, by Luis Landero)

  • /viaf_search/QUERY

    For a given plain text search query, returns the VIAF search results.

    Example

  • /initial_search/QUERY

    For a given human name with initials, find Wikidata items with labels/aliases that match the name with expanded initials.
